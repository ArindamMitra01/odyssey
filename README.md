# 🛡️ Agentic Finance Pilot

An AI-driven fintech dashboard designed to analyze personal purchasing power and optimize market investments through automated agentic reasoning.

Access the live application here:
**[👉 Launch Agentic Finance Pilot](https://odyssey-akmfquvhjhaylq2hgqxuo3.streamlit.app/)**

## 🛠️ Project Architecture
This project utilizes a dual-engine analytical framework to bridge the gap between personal transactions and market realities, hosted entirely via **Streamlit**.

- **GWTS (Goal-Weighted Ticker Sieve):** A proprietary ranking engine that filters NSE equities by prioritizing value signatures over market volatility.
- **PBV (Personalized Basket Variance):** A real-world inflation engine that maps user spending to sector-specific indices to calculate a personal CPI.
- **Agentic Brain:** Powered by **Gemini 2.5-Flash-Lite**, providing high-speed, bulleted executive summaries and strategic investment hedges.
